# Web Starter

Simple web starter with  Webpack, Babel, PostCSS, BrowserSync...

## Getting Started

```bash
git clone git@github.com:vn38minhtran/color-name.git example
cd example
npm install && bower install
npm start
```

## Build

```bash
npm run build
```
